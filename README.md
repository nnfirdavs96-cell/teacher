# Мастер ТВ/Сети 📡

Мобильное приложение-помощник для мастеров по **установке и ремонту**
телевидения и домашних сетей. Развито из мастер-класса DVB-S2 / АНТ
в полноценную базу знаний и набор рабочих инструментов.

> Исходные материалы (`DVB-S2_ANT_v9 (1).pptx`, `DVB-S2_ANT_Cheatsheet_v9.pdf`)
> сохранены в репозитории как первоисточник.

## Что внутри

| Раздел | Описание |
|--------|----------|
| 📖 **Справочник** | Статьи по сигналу/измерениям, спутнику (DVB-S2), АНТ/MMDS, эфиру (DVB-T2), кабельному (DVB-C), оптике/GPON, Wi-Fi, сетям/СКС, кабелям и разъёмам. Полнотекстовый поиск. |
| 🧮 **Калькуляторы** | Затухание кабеля · Бюджет сигнала крыша→квартира · Конвертер dBm↔dBµV↔мВт · Азимут/элевация/skew на спутник · Пересчёт транспондера в IF (LNB). |
| 🔧 **Диагностика** | Поиск неисправностей по симптомам с пошаговым разбором причин и решений. |
| 🛰️ **Частоты** | Спутники с транспондерами, частотные диапазоны (C/Ku/Ka/L/UHF/VHF), сетка каналов DVB-T2, каналы Wi-Fi. |
| ✅ **Чеклисты** | Пошаговые алгоритмы наводки и монтажа с отметкой выполнения. |
| 🤖 **ИИ-помощник** | Вопрос своими словами — отвечает с учётом норм и практики мастера. |

## Технологии

- **Expo** (React Native + TypeScript) + **expo-router** — один код для
  **Android, iOS и веб (PWA)**.
- Вся справочная часть, калькуляторы, диагностика и чеклисты работают
  **офлайн** — удобно на крыше и в подвале без интернета.
- ИИ-помощник требует интернета и API-ключа Anthropic (хранится только
  на устройстве).

## Запуск

```bash
npm install
npm start          # затем отсканируй QR в Expo Go (Android/iOS)
npm run android    # эмулятор Android
npm run ios        # симулятор iOS (macOS)
npm run web        # версия в браузере / PWA
npm run typecheck  # проверка типов
```

### Сборка APK для телефона

```bash
npm install -g eas-cli
eas build -p android --profile preview
```

## ИИ-помощник

Открой вкладку **ИИ → ⚙️** и вставь API-ключ Anthropic
(console.anthropic.com). Ключ сохраняется в защищённом хранилище
устройства (SecureStore), на веб — в локальном хранилище браузера.
Модель переключается там же (Sonnet / Opus / Haiku).

## Структура

```
app/                # экраны (expo-router)
  (tabs)/           # вкладки: справочник, калькуляторы, диагностика, частоты, чеклисты, ИИ
  article/[id].tsx  # карточка статьи справочника
data/               # офлайн-база: знания, спутники, диагностика, чеклисты
lib/                # calc.ts (расчёты), ai.ts (клиент ИИ)
components/ui.tsx   # переиспользуемые компоненты
constants/theme.ts  # тёмная тема
```

## Оговорка

Числовые нормы и частоты — типовые ориентиры для монтажа. Конкретный
оператор может задавать свои пороги; частоты транспондеров сверяйте с
актуальными источниками (например lyngsat.com).
